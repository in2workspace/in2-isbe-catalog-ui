# Authors

Portal del Catálogo de Servicios Comerciales de ISBE es el resultado de esfuerzos colaborativos de múltiples organizaciones e individuos.

## Primary Development

**Altia Consultores, S.A.** (2021-2025)

- Desarrollo principal y arquitectura

## Project Sponsors

**Alastria** (2019-present)

- ISBE Project governance and coordination

**Comunidad Autónoma de Madrid (CAM)** (2025-present)

- Patrocinio y apoyo al proyecto

## Contribuidores individuales

Además de los colaboradores organizacionales mencionados anteriormente, este proyecto
se ha beneficiado de las contribuciones de desarrolladores individuales. Se puede encontrar una lista completa
en el historial de commits de Git.

Para ver los contribuyentes individuales:

```text
git log --format='%aN <%aE>' | sort -u
```
